"""
This project involves developing a solver for Nonograms (also known as Picross, Griddlers, or Japanese crosswords), 
which are logic puzzles thatconsist of a grid of cells that must be filled in or left blank according to numeric
clues provided for each row and column. The goal is to discover a hidden image within the grid by correctly
interpreting these clues.

Each row and column is associated with a set of clues, which indicate the lengths of consecutive sequences
of filled cells. For example, a clue of [3, 1] means there are three consecutive filled cells followed by
at least one empty cell, and then another single filled cell.

Features of the project: 
- Image to Nonogram: Convert an image to a Nonogram puzzle
- Nonogram Solver: Solve a Nonogram puzzle to reveal the hidden image
- Nonogram to Image: Convert a solved Nonogram puzzle back to an image
- Nonogram Generator: Generate a random Nonogram puzzle

Image to Nonogram:
Convert an image to a Nonogram puzzle by converting the image to grayscale,
analysing pixel values and applying a threshold to determine filled and
blank cells.

Nonogram Solver:
Solve a Nonogram puzzle using a constraint satisfaction algorithm that
iteratively applies logical deductions to determine which cells are filled
and which cells are blank. If a unique solution exists, the solver will
eventually find it. If multiple solutions exist, the solver will find one
of them.
"""
